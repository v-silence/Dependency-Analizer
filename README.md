# Визуализатор зависимостей Python
Данный инструмент командной строки предназначен для визуализации графа зависимостей Python пакетов, включая транзитивные зависимости. Визуализатор использует язык описания графов PlantUML для создания изображения, которое сохраняется в файл формата PNG.

## Описание
Визуализатор зависимостей анализирует указанный Python пакет и строит граф зависимостей на основе его зависимостей. Глубина анализа зависимостей может быть задана в конфигурационном файле. Результат визуализации сохраняется в файл формата PNG, заданный в конфигурационном файле.
Визуализатор не использует сторонние средства для получения зависимостей, а определяет их по имени пакета.

## Требования
Для работы визуализатора необходимо наличие следующих компонентов:

- Python 3.6 или новее
- tomli
- pytest (для запуска тестов)
## Установка
- ### Клонируйте репозиторий с проектом:
git clone https://github.com/v-silence/Dependency-Analizer.git
- ### Перейдите в директорию проекта:
cd Dependency-Analizer
- ### Установите необходимые зависимости
##  Использование
После успешного завершения работы визуализатора, изображение графа зависимостей будет сохранено в указанном файле.
